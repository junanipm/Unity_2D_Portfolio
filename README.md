변신 시스템을 가미한 전통적인 2D 액션 플랫포머 게임입니다.

보스를 처치하며 세상에 색과 빛을 되찾아가는 모험을 떠납니다.
색을 되찾아 갈 때 마다 새로운 무기와 스킬을 얻을 수 있으며, 이를 통해 상황과 플레이어의 취향에 맞는 다양한 액션을 느낄 수 있습니다.
